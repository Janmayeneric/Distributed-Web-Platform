About the System


This is our implementation for a federated Reddit, which allows an individual to create an account, post content, interact with communities and other users through notifications, comments and follower requests. Since our system is only a single node in the federated network, it must also allow users to view and share content from other implementations via a communication protocol.
Creators : Josh Wheeler, Adin Johnstone, Xinyu Hu, Matthew Foulds.
